# Sprint 1 Plan – StudyBuddy Task Manager

## Sprint Goal
By the end of Sprint 1, we will have a **basic StudyBuddy Task Manager** where a student can:
- Add a task
- See a list of tasks
- Mark a task as complete

---

## User Stories in Sprint 1 (Must Have)

- [ ] As a student, I want to **add a task**, so that I can remember my homework.
- [ ] As a student, I want to **see a list of all my tasks**, so that I know what I need to do.
- [ ] As a student, I want to **mark a task as complete**, so that I know what I’ve already finished.

---

## Tasks for This Sprint

### Feature: Add Task
- [ ] Create UI/form for entering a new task (task name, optional due date).
- [ ] Add a button to submit the task.
- [ ] Store the new task in a list/array/data structure.
- [ ] Clear the input field after adding.

### Feature: View Task List
- [ ] Display all saved tasks in a list.
- [ ] Show basic details (task name, completion status).
- [ ] Make sure the list updates immediately after a new task is added.
